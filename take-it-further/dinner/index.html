What's For Dinner?

Do you ever open the fridge or pantry and just stare at all the food inside because you aren't sure what to make? No more! This application will decide what you should eat for dinner!

Build Instructions

  • You should have a different number of main dishes, side dishes and desserts available.
  • Allow the user to click a button to get a new meal - and change the text of the button once they've already generated a first meal
  • Make sure your site has a mobile friendly design
  • Create both a light and a dark layout and set CSS to default to whichever layout matches the user's browser/operating system preference
  • For practice, setup this website as a Github pages site
